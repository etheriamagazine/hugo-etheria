# hugo-etheria

A Hugo module-based theme for Etheria Magazine.


Copyright


## Building & serving site

Starting server & watch with example content:
```
npm run dev
```


Build:
```
npm run build:css && hugo
```

Build site
```
npm run watch:css & hugo server ; fg
```
